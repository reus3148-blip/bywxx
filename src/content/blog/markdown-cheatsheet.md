매번 까먹어서 직접 정리해두는 마크다운 문법 모음.

## 글자 꾸미기

- **굵게** — `**굵게**`
- *기울임* — `*기울임*`
- ~~취소선~~ — `~~취소선~~`
- `인라인 코드` — `` `인라인 코드` ``

## 제목

```markdown
# H1
## H2
### H3
#### H4
```

## 리스트

```markdown
- 글머리 1
- 글머리 2
  - 들여쓰기 2칸이면 하위 리스트

1. 번호 1
2. 번호 2
```

## 링크와 이미지

```markdown
[링크 텍스트](https://example.com)
![대체 텍스트](이미지경로.jpg)
```

## 인용구

```markdown
> 인용할 내용
> 여러 줄도 가능
```

## 코드 블록

````markdown
```js
const hello = '안녕';
console.log(hello);
```
````
